# Paul Jeggels Designs

A modern, responsive surfboard manufacturer website rebuilt with React, Vite, and Tailwind CSS.

## Project Structure

- `src/components`: Reusable UI components (Hero, etc.)
- `src/App.jsx`: Main application entry component
- `src/index.css`: Global styles and Tailwind configuration

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Design Notes

-   **Visual Identity**: Artisanal, deep blues (`#0a192f`), white text (`#e6f1ff`).
-   **Hero Section**: Floating card interface inspired by Kargo, featuring "Request a Quote".
